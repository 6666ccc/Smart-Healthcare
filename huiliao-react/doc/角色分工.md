# 慧疗 (HuiLiao) 前端角色分工与功能矩阵

> 基于 `doc/API.md` v2.0 与 `../HuiLiao/docs/API-按角色分工.md`
> 前端三门户：患者端 / 医生端 / 工作人员端（管理端，含 admin、cashier、pharmacist）

---

## 1. 角色与门户

| 门户 | portalType | 子角色 | 说明 |
|------|------------|--------|------|
| 患者端 | `patient` | — | 患者自助：挂号、查看处方、支付、AI 咨询 |
| 医生端 | `doctor` | — | 医生诊疗：接诊、开方、开检查、录入诊断 |
| 工作人员端 | `admin` | `admin` / `cashier` / `pharmacist` | 院内管理：建档、排班、收费、发药、库存、统计 |

### 子角色菜单差异（工作人员端）

| 功能模块 | admin（管理员） | cashier（收费员） | pharmacist（药师） |
|----------|:---:|:---:|:---:|
| 首页仪表盘 | ✅ | ✅ | ✅ |
| 挂号管理 | ✅ | ✅ | — |
| 接诊管理 | ✅ | — | — |
| 收费管理 | ✅ | ✅ | — |
| 发药管理 | ✅ | — | ✅ |
| 患者管理 | ✅ | — | — |
| 排班管理 | ✅ | — | — |
| 药品管理 | ✅ | — | ✅ (查看) |
| 科室查询 | ✅ | ✅ | ✅ |
| 库存管理 | — | — | ✅ |

## 2. 患者端功能详表

| 功能 | 路径 | API | 说明 |
|------|------|-----|------|
| 个人档案 | `/user` | GET/PUT `/api/patients/{id}` | 查看/更新个人信息 |
| 科室查询 | `/department` | GET `/api/depts` | 浏览科室列表 |
| 医生查询 | (科室详情) | GET `/api/staff` | 查看医生列表 |
| 预约挂号 | `/registration` | POST `/api/registrations` | 选择排班挂号 |
| 挂号记录 | `/registration` | GET `/api/registrations` | 查看/取消挂号 |
| 就诊详情 | `/registration/:id` | GET `/api/visits/{id}` | 查看诊断结果 |
| 处方查看 | — (待增强) | GET `/api/prescriptions` | 查看处方与用药说明 |
| 检查报告 | — (待增强) | GET `/api/exam-requests` | 查看检查申请 |
| 门诊缴费 | `/payment` | GET `/api/charges`, POST `/api/charges/{id}/pay` | 查看待缴费/支付 |
| AI 助手 | `/assistant` | POST `/api/ai/chat` | 智能问诊咨询（仅患者端） |

## 3. 医生端功能详表

| 功能 | 路径 | API | 说明 |
|------|------|-----|------|
| 工作台 | `/home` | GET `/api/dashboard` | 待诊统计 |
| 接诊管理 | `/consultation` | POST `/api/visits/start/{regId}`, PUT `/api/visits/{id}` | 开始接诊/录入诊断 |
| 处方开立 | `/consultation` | POST `/api/prescriptions` | 选择药品开处方 |
| 检查申请 | `/consultation` | POST `/api/exam-requests` | 开立检查申请 |
| 排班查看 | `/registration` | GET `/api/schedules` | 查看排班 |
| 检查报告 | — (待新增) | GET `/api/exam-requests` | 查看已开检查 |
| 患者查询 | — (接诊内) | GET `/api/patients` | 查询患者信息 |

## 4. 工作人员端功能详表

### 4.1 管理员 (admin)

| 功能 | 路径 | API |
|------|------|-----|
| 仪表盘 | `/home` | GET `/api/dashboard` |
| 挂号管理 | `/registration` | GET/POST `/api/registrations`, POST `/api/registrations/{id}/cancel` |
| 接诊管理 | `/consultation` | GET `/api/visits` |
| 收费管理 | `/payment` | GET `/api/charges`, POST `/api/charges/from-visit/{visitId}`, POST `/api/charges/{id}/pay` |
| 发药管理 | `/dispense` | GET `/api/prescriptions/pending-dispense`, POST `/api/dispense/{prescriptionId}` |
| 患者管理 | `/patients` | GET/POST/PUT `/api/patients` |
| 排班管理 | `/schedules` | GET/POST/PUT `/api/schedules` |
| 药品管理 | `/drugs` | GET/POST/PUT `/api/drugs` |
| 科室查询 | `/department` | GET `/api/depts` |

### 4.2 收费员 (cashier)

| 功能 | 路径 | API |
|------|------|-----|
| 现场挂号 | `/registration` | POST `/api/registrations` |
| 收费管理 | `/payment` | GET `/api/charges/pending`, POST `/api/charges/from-visit/{visitId}`, POST `/api/charges/{id}/pay` |

### 4.3 药师 (pharmacist)

| 功能 | 路径 | API |
|------|------|-----|
| 发药管理 | `/dispense` | GET `/api/prescriptions/pending-dispense`, POST `/api/dispense/{prescriptionId}` |
| 药品库存 | — (Dashboard 内) | GET `/api/drug-stocks?lowStockOnly=true` |

## 5. 门诊闭环流程（前端交互）

```
患者端                  医生端                  工作人员端
─────                  ──────                  ────────
                                                    【管理员 排班】
                                                    POST /api/schedules

【查看排班】            【查看排班】
GET /api/schedules      GET /api/schedules

【自助建档】                                    【现场建档】
POST /api/patients                              POST /api/patients

【自助挂号】                                    【现场挂号】
POST /api/registrations                         POST /api/registrations

                        【查看待诊】
                        GET /api/registrations/pending

                        【开始接诊】
                        POST /api/visits/start/{id}

                        【录入诊断】
                        PUT /api/visits/{id}

                        【开处方/开检查】
                        POST /api/prescriptions
                        POST /api/exam-requests

                                                    【收费员 生成收费单】
                                                    POST /api/charges/from-visit/{visitId}

【自助支付】                                    【窗口收费】
POST /api/charges/{id}/pay                      POST /api/charges/{id}/pay

                                                    【药师 发药】
                                                    POST /api/dispense/{id}

【查看处方】            【查看处方】
GET /api/prescriptions   GET /api/prescriptions

【AI 用药咨询】
POST /api/ai/chat
```
